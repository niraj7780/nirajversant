 🎯 NirajVersant

A free English proficiency practice platform built for learners preparing for Versant-style speaking and writing assessments.

🚀 Overview

NirajVersant provides realistic Versant-format practice tests with AI-powered scoring and detailed feedback to help users improve their English communication skills.

Platform Features

* ✅ 15 Full Versant-Style Practice Tests
* 🎤 Speaking Assessment Modules
* ⌨️ Typing & Dictation Exercises
* 📝 Passage Reconstruction Practice
* 🤖 AI-Based Instant Scoring
* 📊 Detailed Performance Feedback
* 📚 Sentence Library
* ⚡ Quick Practice Mode
* 📱 Responsive Design
* 🔒 Privacy-Focused Experience

 📋 Test Structure

| Part | Section                | Type          | Items | Points |
| ---- | ---------------------- | ------------- | ----- | ------ |
| A    | Repeat                 | Speaking      | 16    | 16     |
| B    | Sentence Builds        | Speaking      | 8     | 8      |
| C    | Conversations          | Speaking      | 12    | 12     |
| D    | Sentence Completion    | Typing        | 18    | 18     |
| E    | Dictation              | Listen + Type | 14    | 14     |
| F    | Passage Reconstruction | Read + Write  | 2     | 12     |

 Total

* Questions: 70
* Duration: ~30 Minutes
* Maximum Score: 80
